# تعليمات التنصيب — مراقب أخبار العراق (عبر GitHub بالكامل)

هذا المسار ما يحتاج الماك يكون مفتوح. كل شي يشتغل على سيرفرات GitHub مجاناً.

## 1) أنشئ مستودع (Repository) جديد
- روح لـ github.com → New Repository
- اسمه مثلاً: `iraq-news`
- خله Public (ضروري حتى يشتغل GitHub Pages مجاناً)

## 2) ارفع الملفات
ارفع هاي الملفات بنفس البنية بالضبط (اسحب وأفلت من "Add file → Upload files"):

```
iraq_news_watcher.py
requirements.txt
.github/workflows/update-news.yml
```

ملاحظة: لما ترفع ملف الـ workflow، لازم يكون بمسار `.github/workflows/update-news.yml` بالضبط. واجهة الرفع بـ GitHub تخليك تكتب المسار الكامل وهي تسوي المجلدات تلقائياً.

## 3) فعّل صلاحية الكتابة لـ Actions
- روح لإعدادات المستودع: Settings → Actions → General
- انزل لـ "Workflow permissions"
- اختر "Read and write permissions"
- احفظ (Save)

هذا ضروري حتى يقدر الـ Bot يحفظ التحديثات بنفس المستودع.

## 4) شغّله أول مرة يدوياً (اختياري بس مفيد للتجربة)
- روح لتبويب "Actions" بالمستودع
- اختر "تحديث أخبار العراق" من القائمة اليسار
- اضغط "Run workflow"
- انتظر دقيقة وحدة وشوف إذا نجح (علامة ✅) أو فشل (❌ — افتح السجل وابعثلي لقطة إذا صار خطأ)

بعدها راح يشتغل تلقائياً كل 30 دقيقة بدون أي تدخل منك.

## 5) فعّل GitHub Pages لعرض الصفحة
- Settings → Pages
- تحت "Build and deployment" اختر: Deploy from a branch
- Branch: `main` / Folder: `/ (root)`
- احفظ

بعد دقيقة أو دقيقتين، الصفحة تصير متاحة على رابط شبيه بـ:
```
https://<اسم-حسابك>.github.io/iraq-news/iraq_news.html
```

## 6) تنزيلها كـ"تطبيق" على الآيفون
- افتح الرابط أعلاه من Safari
- اضغط زر المشاركة (Share) → "Add to Home Screen"
- تصير أيقونة على شاشتك الرئيسية، وكل ما تفتحها تشوف آخر تحديث (يتحدث تلقائياً كل 30 دقيقة بالخلفية عبر GitHub)

## ملاحظات مهمة
- **الحد الأدنى الفعلي:** GitHub ما يضمن دقة الوقت 100% وقت الضغط الكبير على سيرفراتهم، فأحياناً يتأخر التحديث بضع دقائق — طبيعي وما يحتاج قلق.
- **توقف تلقائي بعد 60 يوم خمول:** إذا المستودع ما شافه أي تفاعل (commit) لمدة 60 يوم، GitHub يوقف الجدولة الزمنية تلقائياً. بما إن السكريبت نفسه يسوي commit كل 30 دقيقة، هذا مستبعد يصير طالما شغال.
- **إذا موقع إخباري غيّر تصميمه** وتوقف عن جلب أخبار جديدة: افتح ملف `watcher.log` من داخل المستودع (يتحدث تلقائياً بكل تشغيلة) وابعثلي محتواه، وأصلح الـ selector فوراً.
- ما راح تحتاج تلمس Terminal أو git محلياً أبداً — كل التحديث والرفع يصير تلقائياً جوا GitHub.
